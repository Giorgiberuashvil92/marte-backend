# ­ЪДф рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃћрЃЉрЃўрЃА рЃбрЃћрЃАрЃбрЃўрЃарЃћрЃЉрЃўрЃА рЃњрЃќрЃљрЃЏрЃЎрЃЋрЃџрЃћрЃЋрЃў

## 1. API-рЃА рЃбрЃћрЃАрЃбрЃўрЃарЃћрЃЉрЃљ (Backend)

### Backend-рЃўрЃА рЃњрЃљрЃерЃЋрЃћрЃЉрЃљ
```bash
cd marte-backend
npm run start:dev
```

Backend рЃњрЃљрЃћрЃерЃЋрЃћрЃЉрЃљ `http://localhost:3000`-рЃќрЃћ (рЃљрЃю рЃърЃЮрЃарЃбрЃќрЃћ рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃњрЃљрЃЦрЃЋрЃЌ рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃўрЃарЃћрЃЉрЃБрЃџрЃў).

### API Endpoints-рЃўрЃА рЃбрЃћрЃАрЃбрЃўрЃарЃћрЃЉрЃљ

#### 1. рЃДрЃЋрЃћрЃџрЃљ рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃўрЃА рЃЏрЃўрЃдрЃћрЃЉрЃљ
```bash
curl http://localhost:3000/categories
```

рЃљрЃю browser-рЃерЃў:
```
http://localhost:3000/categories
```

#### 2. рЃърЃЮрЃърЃБрЃџрЃљрЃарЃБрЃџрЃў рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃћрЃЉрЃўрЃА рЃЏрЃўрЃдрЃћрЃЉрЃљ
```bash
curl http://localhost:3000/categories/popular?limit=6
```

#### 3. рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃћрЃЉрЃўрЃА рЃЏрЃўрЃдрЃћрЃЉрЃљ
```bash
curl http://localhost:3000/categories/main
```

#### 4. рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџрЃў рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃўрЃА рЃЏрЃўрЃдрЃћрЃЉрЃљ (ID-рЃўрЃЌ)
```bash
curl http://localhost:3000/categories/{categoryId}
```

#### 5. рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ (POST)
```bash
curl -X POST http://localhost:3000/categories \
  -H "Content-Type: application/json" \
  -d '{
    "name": "рЃбрЃћрЃАрЃб рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃљ",
    "nameEn": "Test Category",
    "description": "рЃбрЃћрЃАрЃбрЃўрЃАрЃЌрЃЋрЃўрЃА",
    "icon": "star",
    "color": "#FF5733",
    "image": "https://images.unsplash.com/photo-1558618666-fcd25c85cd64?q=80&w=1000",
    "isActive": true,
    "order": 10,
    "serviceTypes": ["test"]
  }'
```

#### 6. рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃўрЃА рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃљ (PATCH)
```bash
curl -X PATCH http://localhost:3000/categories/{categoryId} \
  -H "Content-Type: application/json" \
  -d '{
    "name": "рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃБрЃџрЃў рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃљ",
    "isActive": false
  }'
```

#### 7. рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃўрЃА рЃгрЃљрЃерЃџрЃљ (DELETE)
```bash
curl -X DELETE http://localhost:3000/categories/{categoryId}
```

### MongoDB-рЃерЃў рЃерЃћрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃљ

# MongoDB Compass-рЃерЃў рЃљрЃю mongo shell-рЃерЃў

## 2. рЃљрЃЊрЃЏрЃўрЃю рЃърЃљрЃюрЃћрЃџрЃўрЃА рЃбрЃћрЃАрЃбрЃўрЃарЃћрЃЉрЃљ

### рЃљрЃЊрЃЏрЃўрЃю рЃърЃљрЃюрЃћрЃџрЃўрЃА рЃњрЃљрЃерЃЋрЃћрЃЉрЃљ
```bash
cd free-nextjs-admin-dashboard
npm run dev
```

рЃљрЃЊрЃЏрЃўрЃю рЃърЃљрЃюрЃћрЃџрЃў рЃњрЃљрЃћрЃерЃЋрЃћрЃЉрЃљ `http://localhost:3000`-рЃќрЃћ (рЃљрЃю рЃАрЃ«рЃЋрЃљ рЃърЃЮрЃарЃбрЃќрЃћ).

### рЃбрЃћрЃАрЃбрЃўрЃарЃћрЃЉрЃўрЃА рЃюрЃљрЃЉрЃўрЃ»рЃћрЃЉрЃў:

1. **рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃћрЃЉрЃўрЃА рЃњрЃЋрЃћрЃарЃЊрЃўрЃА рЃњрЃљрЃ«рЃАрЃюрЃљ**
   - рЃњрЃљрЃЊрЃљрЃЊрЃўрЃЌ: `http://localhost:3000/categories`
   - рЃљрЃю рЃЏрЃћрЃюрЃўрЃБрЃЊрЃљрЃю: Admin Рєњ Categories

2. **рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃћрЃЉрЃўрЃА рЃАрЃўрЃўрЃА рЃюрЃљрЃ«рЃЋрЃљ**
   - рЃБрЃюрЃЊрЃљ рЃЊрЃљрЃўрЃюрЃљрЃ«рЃЮрЃЌ 16 рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃљ (8 рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў + 8 parts subcategories)
   - рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЌ рЃарЃЮрЃЏ рЃДрЃЋрЃћрЃџрЃљ рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃљ рЃЕрЃљрЃюрЃА

3. **рЃљрЃ«рЃљрЃџрЃў рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃўрЃА рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃљ**
   - рЃЊрЃљрЃљрЃГрЃўрЃарЃћрЃЌ "+ рЃљрЃ«рЃљрЃџрЃў рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃљ" рЃдрЃўрЃџрЃљрЃЎрЃА
   - рЃерЃћрЃљрЃЋрЃАрЃћрЃЌ рЃцрЃЮрЃарЃЏрЃљ:
     - рЃАрЃљрЃ«рЃћрЃџрЃў (рЃЦрЃљрЃарЃЌрЃБрЃџрЃў): "рЃбрЃћрЃАрЃб рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃљ"
     - рЃАрЃљрЃ«рЃћрЃџрЃў (рЃўрЃюрЃњрЃџрЃўрЃАрЃБрЃарЃў): "Test Category"
     - рЃљрЃдрЃгрЃћрЃарЃљ: "рЃбрЃћрЃАрЃбрЃўрЃАрЃЌрЃЋрЃўрЃА"
     - рЃљрЃўрЃарЃЕрЃўрЃћрЃЌ рЃўрЃЎрЃЮрЃюрЃљ
     - рЃљрЃўрЃарЃЕрЃўрЃћрЃЌ рЃцрЃћрЃарЃў
     - рЃЊрЃљрЃљрЃДрЃћрЃюрЃћрЃЌ рЃАрЃћрЃарЃЋрЃўрЃАрЃўрЃА рЃбрЃўрЃърЃћрЃЉрЃў
   - рЃЊрЃљрЃљрЃГрЃўрЃарЃћрЃЌ "рЃерЃћрЃюрЃљрЃ«рЃЋрЃљ"
   - рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЌ рЃарЃЮрЃЏ рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃљ рЃЊрЃљрЃћрЃЏрЃљрЃбрЃљ рЃАрЃўрЃљрЃерЃў

4. **рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃўрЃА рЃарЃћрЃЊрЃљрЃЦрЃбрЃўрЃарЃћрЃЉрЃљ**
   - рЃљрЃўрЃарЃЕрЃўрЃћрЃЌ рЃарЃЮрЃЏрЃћрЃџрЃўрЃЏрЃћ рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃљ
   - рЃЊрЃљрЃљрЃГрЃўрЃарЃћрЃЌ РюЈ№ИЈ (рЃарЃћрЃЊрЃљрЃЦрЃбрЃўрЃарЃћрЃЉрЃљ) рЃдрЃўрЃџрЃљрЃЎрЃА
   - рЃерЃћрЃфрЃЋрЃљрЃџрЃћрЃЌ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў
   - рЃЊрЃљрЃљрЃГрЃўрЃарЃћрЃЌ "рЃерЃћрЃюрЃљрЃ«рЃЋрЃљ"
   - рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЌ рЃарЃЮрЃЏ рЃфрЃЋрЃџрЃўрЃџрЃћрЃЉрЃћрЃЉрЃў рЃерЃћрЃўрЃюрЃљрЃ«рЃљ

5. **рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃўрЃА рЃљрЃЦрЃбрЃўрЃЋрЃљрЃфрЃўрЃљ/рЃЊрЃћрЃљрЃЦрЃбрЃўрЃЋрЃљрЃфрЃўрЃљ**
   - рЃЊрЃљрЃљрЃГрЃўрЃарЃћрЃЌ ­ЪЉЂ№ИЈ (toggle active) рЃдрЃўрЃџрЃљрЃЎрЃА
   - рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЌ рЃарЃЮрЃЏ рЃАрЃбрЃљрЃбрЃБрЃАрЃў рЃерЃћрЃўрЃфрЃЋрЃљрЃџрЃљ

6. **рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃўрЃА рЃгрЃљрЃерЃџрЃљ**
   - рЃЊрЃљрЃљрЃГрЃўрЃарЃћрЃЌ ­ЪЌЉ№ИЈ (рЃгрЃљрЃерЃџрЃљ) рЃдрЃўрЃџрЃљрЃЎрЃА
   - рЃЊрЃљрЃљрЃЊрЃљрЃАрЃбрЃБрЃарЃћрЃЌ рЃгрЃљрЃерЃџрЃљ
   - рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЌ рЃарЃЮрЃЏ рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃљ рЃгрЃљрЃўрЃерЃљрЃџрЃљ

## 3. Parts рЃњрЃЋрЃћрЃарЃЊрЃўрЃА рЃбрЃћрЃАрЃбрЃўрЃарЃћрЃЉрЃљ (рЃљрЃЊрЃЏрЃўрЃю рЃърЃљрЃюрЃћрЃџрЃерЃў)

1. **Parts рЃњрЃЋрЃћрЃарЃЊрЃўрЃА рЃњрЃљрЃ«рЃАрЃюрЃљ**
   - рЃњрЃљрЃЊрЃљрЃЊрЃўрЃЌ: `http://localhost:3000/parts`
   - рЃљрЃю рЃЏрЃћрЃюрЃўрЃБрЃЊрЃљрЃю: Admin Рєњ Parts

2. **рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃћрЃЉрЃўрЃА Dropdown-рЃўрЃА рЃерЃћрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃљ**
   - рЃЊрЃљрЃљрЃГрЃўрЃарЃћрЃЌ "+ рЃљрЃ«рЃљрЃџрЃў Part" рЃдрЃўрЃџрЃљрЃЎрЃА
   - рЃњрЃљрЃЊрЃљрЃЊрЃўрЃЌ "Category" dropdown-рЃќрЃћ
   - рЃБрЃюрЃЊрЃљ рЃЊрЃљрЃўрЃюрЃљрЃ«рЃЮрЃЌ parts-рЃўрЃА рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃћрЃЉрЃў:
     - рЃФрЃарЃљрЃЋрЃљ
     - рЃбрЃарЃљрЃюрЃАрЃЏрЃўрЃАрЃўрЃљ
     - рЃцрЃљрЃарЃћрЃЉрЃў
     - рЃАрЃљрЃЉрЃБрЃарЃљрЃЋрЃћрЃЉрЃў
     - рЃЉрЃџрЃЮрЃЎ-рЃцрЃљрЃарЃћрЃЉрЃў
     - рЃўрЃюрЃбрЃћрЃарЃўрЃћрЃарЃў
     - рЃћрЃџрЃћрЃЦрЃбрЃарЃЮрЃюрЃўрЃЎрЃљ
     - рЃАрЃ«рЃЋрЃљ

3. **Part-рЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃўрЃЌ**
   - рЃерЃћрЃљрЃЋрЃАрЃћрЃЌ рЃцрЃЮрЃарЃЏрЃљ
   - рЃљрЃўрЃарЃЕрЃўрЃћрЃЌ рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃљ dropdown-рЃўрЃЊрЃљрЃю
   - рЃЊрЃљрЃљрЃГрЃўрЃарЃћрЃЌ "рЃерЃћрЃюрЃљрЃ«рЃЋрЃљ"
   - рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЌ рЃарЃЮрЃЏ part рЃерЃћрЃўрЃЦрЃЏрЃюрЃљ рЃАрЃгрЃЮрЃарЃў рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃўрЃЌ

## 4. рЃЏрЃЮрЃЉрЃљрЃўрЃџ рЃљрЃърЃўрЃА рЃбрЃћрЃАрЃбрЃўрЃарЃћрЃЉрЃљ

### рЃбрЃћрЃАрЃбрЃўрЃарЃћрЃЉрЃўрЃА рЃюрЃљрЃЉрЃўрЃ»рЃћрЃЉрЃў:

1. **рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃћрЃЉрЃўрЃА API-рЃА рЃерЃћрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃљ**
   - рЃњрЃљрЃ«рЃАрЃћрЃюрЃўрЃЌ app-рЃў
   - рЃњрЃљрЃЊрЃљрЃЊрЃўрЃЌ Marketplace рЃљрЃю Services рЃњрЃЋрЃћрЃарЃЊрЃќрЃћ
   - рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃћрЃЉрЃў рЃБрЃюрЃЊрЃљ рЃЕрЃљрЃўрЃбрЃЋрЃўрЃарЃЌрЃЮрЃА API-рЃЊрЃљрЃю

2. **рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃўрЃА рЃњрЃЋрЃћрЃарЃЊрЃўрЃА рЃбрЃћрЃАрЃбрЃўрЃарЃћрЃЉрЃљ**
   - рЃЊрЃљрЃљрЃГрЃўрЃарЃћрЃЌ рЃарЃЮрЃЏрЃћрЃџрЃўрЃЏрЃћ рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃљрЃА
   - рЃњрЃљрЃЊрЃљрЃЊрЃўрЃЌ `/category?type=carwash` (рЃљрЃю рЃАрЃ«рЃЋрЃљ рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃљ)
   - рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЌ рЃарЃЮрЃЏ:
     - рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃўрЃА рЃАрЃљрЃ«рЃћрЃџрЃў рЃАрЃгрЃЮрЃарЃљрЃЊ рЃЕрЃљрЃюрЃА
     - рЃўрЃЎрЃЮрЃюрЃљ рЃЊрЃљ рЃцрЃћрЃарЃў рЃАрЃгрЃЮрЃарЃўрЃљ
     - рЃАрЃћрЃарЃЋрЃўрЃАрЃћрЃЉрЃў рЃЕрЃљрЃўрЃбрЃЋрЃўрЃарЃЌрЃћрЃЉрЃљ

3. **API-рЃЊрЃљрЃю рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃћрЃЉрЃўрЃА рЃЕрЃљрЃбрЃЋрЃўрЃарЃЌрЃЋрЃљ**

**рЃњрЃљрЃЊрЃљрЃгрЃДрЃЋрЃћрЃбрЃљ:**

**рЃњрЃљрЃЊрЃљрЃгрЃДрЃЋрЃћрЃбрЃљ:**

**рЃњрЃљрЃЊрЃљрЃгрЃДрЃЋрЃћрЃбрЃљ:**

3. рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЌ рЃарЃЮрЃЏ `categoriesApi.ts` рЃАрЃгрЃЮрЃарЃљрЃЊ рЃўрЃЏрЃърЃЮрЃарЃбрЃўрЃарЃћрЃЉрЃБрЃџрЃўрЃљ

**рЃњрЃљрЃЊрЃљрЃгрЃДрЃЋрЃћрЃбрЃљ:**
